Notes

There are two types of units in CSS

Absolute

px. pt, in, cm, mm

Relative

% relative to the size of the container

vw, vh relative to the viewport

em, rem relative to the font size

Commentary

pixels or px do not change size. Some people say not good because they are not responsive.Mosh = some things should remain a certain size

Benefit of relative elements is that they are responsive
